# Battle effect sprite attribution

The PNG effect sprites in this directory were mirrored from Pokémon Showdown's
public battle effect directory:

- https://play.pokemonshowdown.com/fx/
- https://github.com/smogon/pokemon-showdown-client

Pokémon Showdown's client credits Kevin Lau (Ascriptmaster) for battle animation
art. The application hosts these files locally instead of consuming Pokémon
Showdown's bandwidth at runtime.
